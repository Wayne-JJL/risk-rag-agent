【AUC】
定义：AUC 表示 ROC 曲线下面积，用于衡量模型区分正负样本的能力。
计算方式：通常由模型预测分数与真实标签计算得到，不需要先指定分类阈值。
使用建议：适合评估整体排序能力，但应与阈值类指标一起使用。
注意事项：当样本分布明显变化时，AUC 可能波动，需要结合 PSI 或样本分析一起看。

【KS】
定义：KS 表示正负样本累计分布差异的最大值。
计算方式：基于模型分数排序后，计算正负样本累计分布差，并取最大值。
使用建议：常与 AUC 一起监控，适合风控模型效果评估。
注意事项：KS 下降可能由特征失效、样本漂移、策略变化等多种因素导致。

【PSI】
定义：PSI 用于衡量两组样本在某个变量或样本整体分布上的差异。
计算方式：将样本按区间分箱，比较训练样本与当前样本在各分箱中的占比差异，再进行加总。
使用建议：适合监控样本稳定性和分布漂移。
注意事项：PSI 偏高时，应重点检查渠道变化、人群变化、时间窗口变化和特征分布变化。

【逾期率】
定义：逾期率表示在特定观察期内，发生逾期的账户数或金额占总账户数或总金额的比例。
常见计算口径：
1. 按账户数：逾期账户数 / 总账户数
2. 按金额：逾期金额 / 总应还金额
3. 按逾期层级：如 M1、M2、M3
使用建议：使用时必须明确观察期、统计口径和业务定义。
注意事项：不同部门对逾期率口径可能不同，必须先确认口径一致。

【坏账率】
定义：坏账率表示坏账账户数或坏账金额占总账户数或总金额的比例。
常见计算口径：
1. 按账户数：坏账账户数 / 总账户数
2. 按金额：坏账金额 / 总放款金额或总应还金额
使用建议：需明确坏账定义，例如核销、损失确认或超过一定账龄。
注意事项：坏账率通常具有滞后性，应与逾期率和回收率结合分析。

【通过率】
定义：通过率表示审批通过样本数占总申请样本数的比例。
计算方式：通过样本数 / 总申请样本数
使用建议：常用于监控策略松紧程度和业务承接能力。
注意事项：应区分预审通过率、终审通过率和放款通过率。

【拒绝率】
定义：拒绝率表示审批拒绝样本数占总申请样本数的比例。
计算方式：拒绝样本数 / 总申请样本数
使用建议：适合与通过率、风险表现一起分析。
注意事项：拒绝率升高可能由策略收紧、客群变化或风险偏好调整导致。

【召回率】
定义：召回率表示真实正类样本中被模型识别出来的比例。
计算方式：TP / (TP + FN)
使用建议：在高风险识别场景中，召回率通常是重要指标。
注意事项：召回率提高可能带来误杀增加，需要与精准率一起平衡。

【精准率】
定义：精准率表示被模型判定为正类的样本中，真实为正类的比例。
计算方式：TP / (TP + FP)
使用建议：适用于关注命中准确性的场景。
注意事项：精准率与召回率之间通常存在权衡关系。

【F1】
定义：F1 是精准率与召回率的调和平均数。
计算方式：2 × Precision × Recall / (Precision + Recall)
使用建议：适合做综合评估，尤其在样本不平衡时。
注意事项：F1 只是综合指标，不能替代业务收益和风险表现指标。

【AUC下降同时PSI升高】
常见解释：通常意味着样本分布发生变化，模型面对的当前样本和训练样本不再一致。
优先排查：
1. 渠道变化
2. 用户结构变化
3. 时间窗口变化
4. 特征稳定性
5. 标签质量
使用建议：先看样本分布，再看特征稳定性和标签质量。
