# risk-rag-agent

Risk RAG agent project, cleaned for GitHub upload.

## Structure

- `src/`: application source code
- `src/data/`: metrics and knowledge text files
- `docs/`: project documents
- `screenshots/`: images and UI captures

## Quick Start

1. Copy `.env.example` to `.env` and fill values.
2. Install dependencies: `pip install -r requirements.txt`
3. Build index: `python src/build_rag_index.py`
4. Run app: `python src/app.py`
